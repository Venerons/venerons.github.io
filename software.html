Software Lab

The Software Lab is a laboratory dedicated to the development of software for various electronic devices, such as traditional desktops, laptops, smartphones and tablets, and various operating systems such as Mac OS X, Windows, Linux, iOS, Android, Firefox OS, and others.

The Software Lab is the main laboratory from Venerons Lab*s and is constantly experimenting with new software solutions for bringing innovation and utility.

Projects

CassiaJS

JavaScript Cryptography Library. It provides a huge collection of criptographic, steganographic and key generation algoritms in pure JavaScript and you can also create one of your own.

CassiaPy

Python Cryptography Library. It provides a huge collection of criptographic, steganographic and key generation algoritms in pure Python and you can also create one of your own.

CivJS

JavaScript Civilization-like game project. It implements similar but simpler rules, and thanks to the web technologies is available to almost any modern OS.

iRoller

Dice Systems Simulator, it provides a good collection of game systems such d20 System, Catan dice game, Formula Dé, Memoir'44, BattleLore, C&C Ancients, Battles of Westeros, Tiket to Ride dice game and others.

SoundLab

SoundLab is an integrated sound environment for Android tablets. It features a Synth, a Drum Pad and a Recorder. A Sampler and a Sequencer will be available soon.

Sounder

HTML5 Virtual Drum Pad, developed with PhoneGap.

JavaScript Sandbox

A JavaScript sandbox playground to test and profile in real time your JS scripts.

BolideBoard

Bolide virtual game board, let you play the Bolide boardgame on desktop, laptop or tablet with easy drag & drop system.

VectorQuidditch

SVG Quidditch Game, a remake of the old Broomsticks game using modern web technologies. ACTUALLY ON ALPHA STAGE.
